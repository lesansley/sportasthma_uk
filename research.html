Significant research

The following publication list provides a source for those wishing to learn more about exercise-induced asthma (EIA) in athletes. The list is not intended to be comprehensive but to provide a useful overview of the available literature. Readers are particularly directed to European Respiratory Society Monograph, which provides a detailed overview of this topic.
